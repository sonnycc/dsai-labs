# Data Science - Labs

In this repository you will find the course syllabus and lab assignments for the Data Science course which is part of the study programme *applied computer science* at [HOGENT](https://www.hogent.be).

Lecture slides can be viewed here <https://hogenttin.github.io/dsai-labs/>.

In order to work effectively on the lab exercises, we suggest the following workflow:

## Local setup

- First, create a local clone of this repository. Don't make changes in your local copy!
- Create a separate directory for your own lab notes and solutions (and optionally, initialize a Git repository)
- When you want to work on a specific chapter:
    - Do a git pull of the original repository to get the latest version of the lab assignment and
    - Copy the files for that chapter to your own directory and work on them there.

Don't forget to also copy over the `data/` directory containing the datasets.

You can open the notebooks in VS Code, as is explained in the first lecture of the course.

## Google Colab

If you don't want to set up a local environment, you can also usethe online [Google Colab](https://colab.research.google.com) environment. It does not require you to install additional software, but you will need a Google account and a Google Drive to store your files.

To get started, first create an empty notebook on Colab (that you can later use to experiment or to keep scratch notes). Your files will be stored in your Google Drive in a specific directory (*Colab Notebooks*).

Next, create a local clone of this directory. When you want to work on a specific chapter, do a git pull of the original repository to get the latest version of the lab assignment and upload the files for that chapter to your Google Drive (under the directory Colab Notebooks). Don't forget to also copy over the `data/` directory containing the datasets.
